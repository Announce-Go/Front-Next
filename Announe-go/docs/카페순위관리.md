#### 데이터 처리
- 실시간 순위 조회 API 호출
  - 요청: 키워드, URL
  - 응답: 현재순위
  - 저장: DB 저장 안함
- 추적 목록 조회 API 호출
  - 요청: 페이지, 필터(상태), 검색어
  - 응답: 추적목록, 전체건수

#### 화면 전환
- [조회] 클릭 → 실시간 순위 조회 및 결과 표시
- 추적 목록 row 클릭 → 추적 상세 페이지로 이동 (`/admin/cafe-rank/tracking/:id`)

#### 특이사항
- 실시간 조회는 DB에 저장하지 않음
- 추적 목록은 전체 광고주/업체의 추적 데이터 표시
- 상태 필터: 전체, 진행중, 완료, 중단
- 검색창에서 키워드, 업체명, 광고주명을 통합 검색 가능
- 진행률: 순위가 확인된(노출된) 횟수 / 25회
- 최근순위: 가장 최근에 조회된 순위

#### 데이터 처리
- 추적 상세 조회 API 호출
  - 요청: 추적ID
  - 응답: 추적정보, 회차별순위히스토리
- 추적 중단 API 호출
  - 요청: 추적ID
  - 응답: 처리결과

#### 화면 전환
- [← 목록으로] 클릭 → 카페 순위 관리 페이지로 이동
- [중단] 클릭 → 확인 모달 → 중단 처리 (데이터는 보존)

#### 특이사항
- 회차는 노출된 날짜 25회를 기준으로 자동 구분
- 미노출(순위권 외)인 날은 노출 횟수에 카운트 안됨
- 회차별로 영역을 구분하여 표시 (최신 회차가 상단)
- 각 회차 내에서 날짜는 내림차순 정렬 (최신 날짜가 상단)
- 추적 중단은 관리자만 가능 (데이터는 삭제되지 않고 상태만 변경)
- 자동 크롤링으로 매일 순위 조회 및 저장
