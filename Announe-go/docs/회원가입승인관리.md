#### 데이터 처리
- 신청 목록 조회 API 호출
  - 요청: 페이지, 필터(구분, 상태), 검색어
  - 응답: 신청목록, 전체건수

#### 화면 전환
- row 클릭 → 승인 상세 페이지 (`/admin/signup-requests/:id`)

#### 특이사항
- 기본 필터값: 구분(전체), 상태(승인 대기)
- 광고주는 카테고리 컬럼에 "-" 표시
- 상태: 대기(노란색), 승인(초록색), 반려(빨간색) 등 색상으로 구분 가능
- 상태 필터 옵션: 전체, 승인 대기, 승인, 반려
