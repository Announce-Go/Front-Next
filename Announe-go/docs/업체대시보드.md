#### 데이터 처리
- 대시보드 통계 API 호출
  - 요청: 없음 (세션에서 사용자 식별)
  - 응답: 매핑광고주수, 추적중인항목수, 포스팅기록수, 카테고리별현황
- 최근 추적 현황 목록 조회 API 호출
  - 요청: limit=5 (최근 5건만)
  - 응답: 추적목록

#### 화면 전환
- 좌측 메뉴 클릭 → 각 메뉴에 해당하는 페이지로 이동
- 최근 추적 현황 row 클릭 → 해당 카테고리의 추적 상세 페이지로 이동
- [전체 보기 →] 클릭 → 순위 관리 페이지로 이동

#### 특이사항
- 좌측 네비게이션은 모든 업체 화면에서 공통으로 표시됨
- 현재 선택된 메뉴는 하이라이트 표시 (►)
- 본인 업체에 매핑된 광고주의 데이터만 표시
- 통계는 본인이 등록한 추적 및 포스팅 데이터만 집계
- 최근 추적 현황은 최근 5건만 표시
