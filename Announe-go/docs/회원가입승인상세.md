#### 데이터 처리
- 신청 상세 조회 API 호출
  - 요청: 신청ID
  - 응답: 신청상세정보 (기본정보, 카테고리, 첨부파일)
- 광고주 목록 조회 API 호출 (업체인 경우만)
  - 요청: 페이지, 검색어
  - 응답: 승인된 광고주 목록
- 승인 처리 API 호출
  - 요청: 신청ID, 승인여부, 선택된광고주ID목록(업체인 경우)
  - 응답: 처리결과
- 반려 처리 API 호출
  - 요청: 신청ID
  - 응답: 처리결과

#### 화면 전환
- [← 목록으로] 클릭 → 승인 관리 목록 페이지로 이동
- [승인] 클릭 →
  - 업체: 광고주 미선택 시 경고 메시지, 선택 시 승인 처리 후 목록으로 이동
  - 광고주: 바로 승인 처리 후 목록으로 이동
- [반려] 클릭 → 반려 처리 후 목록으로 이동

#### 특이사항
- 업체 승인 시 광고주 선택은 필수 (최소 1개)
- 광고주는 다중 선택 가능 (N:N 관계)
- 광고주 목록은 "승인 완료" 상태인 광고주만 표시
- 파일 다운로드 및 미리보기 기능 제공
