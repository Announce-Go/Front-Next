#### 데이터 처리
- 포스팅 목록 조회 API 호출
  - 요청: 페이지, 검색어
  - 응답: 포스팅목록, 전체건수

#### 특이사항
- 관리자는 조회만 가능 (등록/수정/삭제는 업체에서 수행)
- 검색창에서 키워드, 업체명, 광고주명을 통합 검색 가능
- 전체 업체의 포스팅 데이터 표시
- 목록에서 모든 정보를 표시하므로 별도 상세 화면 없음
