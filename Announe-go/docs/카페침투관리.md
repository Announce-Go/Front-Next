#### 데이터 처리
- 카페 침투 목록 조회 API 호출
  - 요청: 연월(YYYY-MM)
  - 응답: 해당월의 카페침투목록(날짜별 그룹화)
- 특정 날짜 카페 침투 조회 API 호출
  - 요청: 날짜(YYYY-MM-DD)
  - 응답: 해당 날짜의 카페침투목록
- 카페 침투 등록 API 호출
  - 요청: 날짜, 내용, 광고주ID
  - 응답: 처리결과
- 카페 침투 수정 API 호출
  - 요청: 침투ID, 내용, 광고주ID
  - 응답: 처리결과
- 카페 침투 삭제 API 호출
  - 요청: 침투ID
  - 응답: 처리결과

#### 화면 전환
- [◀] [▶] 클릭 → 이전/다음 달로 이동, 캘린더 갱신
- 캘린더의 날짜 클릭 → 하단에 해당 날짜의 카페 침투 목록 표시
- [+ 추가] 클릭 → 하단에 카페 침투 등록 폼 표시
- [수정] 클릭 → 하단에 카페 침투 수정 폼 표시 (해당 침투 내용 미리 채움)
- [삭제] 클릭 → 확인 모달 → 삭제 처리 → 목록 갱신
- [취소] 클릭 → 하단 폼 닫기
- [저장] 클릭 → 등록/수정 처리 → 폼 닫기 → 목록 갱신

#### 특이사항
- 캘린더에서 카페 침투가 있는 날짜는 건수 표시 (예: 18•3 = 18일에 3건)
- 관리자가 등록/수정/삭제 가능 (별도 페이지 이동 없이 동일 화면에서 처리)
- 업체/광고주는 조회만 가능
- 날짜 클릭 시에만 하단에 해당 날짜의 카페 침투 목록 표시
- [+ 추가] 또는 [수정] 클릭 시 하단에 폼 표시 (목록 아래에 인라인으로 표시)
- 날짜 필드는 클릭한 날짜로 자동 설정
- 광고주 선택은 드롭다운 (승인된 광고주 목록)
